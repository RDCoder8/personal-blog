Starting to Code!

The first step to becoming a programmer is to learn the basics of coding but before that you must have idea of why you want to code. Some wish for a lucrative career in programming but others just want to learn how to code for a personal project. Passion is the key to success either way! But in this blog we want you to know there are fun ways to learn to code no matter what your goals are.

Nowadays learning is both a challenge and a reward. To match that there are a large number of coding games available around the web! Code Combat is an educational video game about software programming. If you want to learn how to code you can play Code Combat and learn how to code in a fun way. Through 11 units the game teaches about coding languages such as Javascript, Python, Coffeescript, and HTML. It is a gret place to start and more importantly it is fun and free!

Check out this quote from their wikipedia page:

"The game was positively reviewed by PC Magazine, won the 2017 SIIA CODiE award for Best Creativity Tool for Students, and has been named a top pick for learning by Common Sense Education."
